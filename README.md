# Lin-Odikko

A powerful LinkedIn profile browser and invitation automater based out of python 3.

## Requirements
- Python 3
- Selenium
- BeautifulSoup
- Firefox
- Geckodriver

## Run Below
pip install selenium
pip install bs4
Also copy 'geckodriver.exe' to Python's executable PATH
